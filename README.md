<h1>HUB Reworked</h1>

A simple tool to calculate the amount of XP you currently have to validate your Epitech Hub module.

## Documentation

Go to [Documentation](./Documentation.md) for more information.

## Contributing

### Dev

Start in development (with watch) mode:

```bash
yarn start
```

### Build

Build the extension:

```bash
yarn build
```
